# TSCP SLSA L3 Hardening Checklist

This checklist must be fully satisfied before claiming SLSA Build Level 3 compliance under the TSCP-SLSA-L3-Profile-v1.

---

## Workflow Control

- [ ] `.github/workflows/*` protected by CODEOWNERS
- [ ] Required reviews enabled on master
- [ ] Branch protection active (TSCP-CANONICAL-CUSTODY ruleset)
- [ ] No workflow mutation without approval
- [ ] `required_signatures` enforcement on master
- [ ] `non_fast_forward` enforcement on master
- [ ] `deletion` enforcement on master

**Status:** ✅ Branch ruleset active with deletion, non_fast_forward, required_signatures
**Remaining:** CODEOWNERS file for workflow directory

---

## Dependency Control

- [ ] `Cargo.lock` committed
- [ ] `cargo build --locked` in build workflow
- [ ] GitHub Action references pinned to full SHA (not `@v4`)
- [ ] Toolchain version recorded in build-policy.json
- [ ] No wildcard dependencies in Cargo.toml

**Status:** ✅ Cargo.lock committed, `--locked` flag in build workflow
**Remaining:** Pin action references to full SHA, record toolchain version

---

## Provenance Control

- [ ] SLSA predicate generated by trusted builder
- [ ] Subject digest matches artifact
- [ ] Workflow identity verified in predicate
- [ ] Attestation signature verified
- [ ] Provenance stored in `attestations/provenance.intoto.jsonl`
- [ ] Builder identity matches expected workflow

**Status:** ✅ SLSA provenance generation via `actions/attest-build-provenance@v1`
**Remaining:** Download provenance attestation as file, verify subject digest in CI

---

## Transparency Control

- [ ] Sigstore certificate verified
- [ ] Rekor inclusion verified
- [ ] Evidence archived in release bundle
- [ ] `sigstore.bundle` present in attestations/
- [ ] `rekor-proof.json` present in attestations/

**Status:** ⚠️ Cosign installed in build workflow, keyless signing attempted
**Remaining:** Enable OIDC token for keyless signing, extract Rekor inclusion proof, verify in enforcement workflow

---

## TSCP Controls

- [ ] Policy evaluation completed (validate_governance.py)
- [ ] Custody decision recorded in `release-attestation.json`
- [ ] Immutable release evidence stored (artifact upload)
- [ ] TSCP verifier exit code == 0
- [ ] TSCP-SLSA-L3-Profile-v1.json present
- [ ] Evidence manifest present and valid

**Status:** ✅ Verifier implemented, profile defined, evidence manifest exists
**Remaining:** Integrate verifier into enforcement workflow as hard gate

---

## Determinism Control

- [ ] Replay validation (build twice, compare digests)
- [ ] Toolchain pinned in `rust-toolchain.toml`
- [ ] Action references pinned
- [ ] `--locked` flag enforced
- [ ] `reproducibility.json` recorded

**Status:** ✅ `--locked` flag, reproducibility.json in build workflow
**Remaining:** Pin toolchain version, implement replay validation job
