# Tag Custody Policy

## Historical Immutable Tags

The following tags are retained as historical release workflow records:

- tscp-test
- tscp-release-test-v1
- tscp-release-test-v2
- tscp-release-test-v3

These tags predate the current custody enforcement model. They are preserved for provenance and historical auditability.

## Current Custody Tags

New release and milestone tags must:
- use signed annotated tags
- point to verified commits
- satisfy repository protection rules
- include required evidence artifacts where applicable
